# OrgFrame Monorepo

This repo now uses npm workspaces.

## Apps

- `apps/orgframe-app`: product application
- `apps/orgframe-web`: marketing/landing website

## Shared packages

- `packages/theme`: shared design tokens and Tailwind preset
- `packages/ui`: shared React UI primitives

## Install

```bash
npm install
```

## Run apps

```bash
npm run dev:app
npm run dev:web
```

## Build apps

```bash
npm run build:app
npm run build:web
```
